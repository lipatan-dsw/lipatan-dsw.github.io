Report / Laporan

Khusus bagi dropshipper / reseller, kami menyediakan halaman khusus untuk Anda. Anda dapat melihat history transaksi secara online. Mulai dari nomor resi dan detail order Anda, jadi tidak perlu repot-repot menanyakan nomor resi Anda. Anda dapat melihatnya di halaman Report ini. Report ini berisi tanggal order, pengirim, penerima, kode barang, size barang, status dan nomor resi.
